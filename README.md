# EMAIL Spam Classification

# Description
In 2000, Enron was one of the largest companies in the United States. By 2002, it had collapsed into bankruptcy due to widespread corporate fraud. The data has been made public and presents a diverse set of email information ranging from internal, marketing emails to spam and fraud attempts.The Enron email dataset contains approximately 500,000 emails generated by employees of the Enron Corporation. I have implemented an SVM Classifier and a Logisitc Regression model to classify the emails, using a TfidfVectorizer to create numerical vectors from the text, ranking words based on their frequency in the dataset.

# Objective
The main objective of this project is to develop a machine learning model that can accurately classify emails as spam or non-spam (ham) using the Enron email dataset. The project involves data preprocessing, feature extraction, model training, and evaluation. Additionally, a user-friendly interface will be created using Gradio to allow users to input email text and receive real-time predictions on whether the email is spam or not.

## Setup
```bash
git clone https://github.com/the-flying-cow/Email-Spam-classification.git
cd Email-Spam-classification
```

Create a virtual environment:
```bash
python -m venv email_venv # .venv or any name you prefer
```
venv\Scripts\activate
```

## Run the project
In your terminal/command prompt, navigate to the root folder and execute the following 

```bash
pip install -r requirements.txt
```

Inside the src folder, run the following

```bash
python main.py # initial training and saving of model
gradio gradio_ui.py # running model in gradio
```
